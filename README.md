#  Pizza Sales SQL Analysis Project 

**Project Overview**
This project involves a comprehensive analysis of a pizza store's sales records using **MySQL**. The goal was to extract actionable insights regarding revenue, customer preferences, and order patterns.

**Tools Used**
* **Database:** MySQL
* **Visualization:** SQL Query Results (Detailed PDF Report included)
* **Design:** Canva (for documentation)

## Repository Structure
* `Pizza_Sales_Analysis_Queries.sql`: Contains 12+ optimized SQL queries (Basic to Advanced).
*  Raw CSV data used for the analysis , also uploaded
* `Pizza Sales Insight Report.pdf`: A visual breakdown of the findings.

## Key Insights
* Identified the **Top 5 most ordered pizza types**.
* Calculated **Cumulative Revenue** across different dates.
* Analyzed **Category-wise sales distribution**.
